Macでカット＆ペーストをする

2014.02.19

Macではカット＆ペーストはできないと思い込んでいたが、実は標準機能で可能だったのでメモ。
カット＆ペーストのためだけに、『XtraFinder』などのアプリをインストールする必要もなかった…
Windowsに慣れていると、ついついMacでもカット＆ペーストを Cmd+x Cmd+v としてしまうが、Macではちょっと勝手が違う。

正確にはコピー＆ムーブ

ショートカットキーは以下。
Cmd + c でコピー
Cmd + Opt + v でムーブ
これで、カット＆ペーストが可能。
