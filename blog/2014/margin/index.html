marginの省略記法

marginの値を省略してコーディングした時の意味合い。

margin: 0; → （上下左右）
margin: 0 0; → （上下）（左右）
margin: 0 0 0; → （上）（左右）（下）
margin: 0 0 0 0; → （上）（右）（下）（左）※時計回り
