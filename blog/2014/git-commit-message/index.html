Git コミットのメッセージ書き方

2014.08.28

Gitではコミットメッセージの形式に関して制約はないが、標準的な形式は以下となる。

  • １行目にコミットの全体的説明を５０字以内で記述
  • ２行目は空行
  • ３行目以降に変更内容の詳細を記述
  • 現在形で記述

参考

https://www.atlassian.com/ja/git/tutorial/git-basics#!commit
